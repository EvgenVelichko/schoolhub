# **App Name**: School Hub

## Core Features:

- Secure Portal Data Synchronization: Safely integrate with nz.ua to parse and display real-time academic data, including grades, homework assignments, and school schedules, within the School Hub interface.
- Dynamic Personalized Dashboard: A customizable home screen displaying essential widgets like 'Next Lesson', 'Latest Grades', and 'Top Players' leaderboard for quick overview.
- Community Knowledge & Support Hub: Facilitate a vibrant community through a general chat and forum, allowing users to share resources, attach images (e.g., solved tests), and filter discussions by subject and class.
- Gamified Contribution & Progress: Implement a gamification system rewarding users with 'coins' for completing homework or helping peers, contributing to user levels and school leaderboards.
- AI-Powered Subject Trivia: Offer an engaging mini-game where an AI tool generates dynamic trivia questions across various school subjects, enhancing learning through interactive challenges.
- Interactive Educational Mini-Games: Beyond trivia, include additional simple mini-games such as a clicker and a basic 2D platformer to make learning fun and rewarding.
- Real-time Notification System: Deliver instant alerts for new grades, upcoming assignments, and messages within the knowledge hub, keeping users informed.

## Style Guidelines:

- Primary color: Vibrant purple (#B061F6), chosen for its energetic and futuristic feel, providing excellent visibility against a dark canvas and reflecting a modern, digital aesthetic.
- Background color: A very dark, subtle purplish-gray (#141118), creating a deep, immersive environment that aligns with the desired glassmorphism and cyberpunk visual themes.
- Accent color: A deep, saturated blue-violet (#2E1FAD), providing a contrasting highlight for interactive elements and key information, complementing the primary purple while adding visual depth.
- Headline font: 'Space Grotesk' (sans-serif) will be used for headlines and titles to convey a modern, technical, and slightly futuristic aesthetic that aligns with the cyberpunk theme.
- Body font: 'Inter' (sans-serif) will be used for all body text, such as forum posts, chat messages, and data displays, ensuring high readability and a clean, objective feel.
- Sleek, minimalistic, and futuristic vector icons will be employed throughout the application, featuring sharp lines and high contrast to complement the digital and cyberpunk visual style.
- The layout will incorporate a sophisticated Glassmorphism design with transparent, frosted panels and widgets, ensuring an immersive, structured, and visually engaging user interface. The design will be fully responsive, providing an optimal experience on both desktop and mobile devices.
- Smooth and fast UI animations will be used for transitions and interactive elements, enhancing user engagement. Expressive celebratory animations will be triggered by positive events, such as achieving high grades or reaching new game levels.
